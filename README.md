# 날씨앱
> 처음으로 만들어본 앱

[SK Weather Planet API](https://openapi.sk.com/api/detailView)를 사용하여 만든 날씨앱

## 설명

- 기간: 2020.02.24 ~ 2020.02.27
- 사용 기술:
  - 언어: Swift
  - 프레임워크: UIKit
- 인원: 1명
- 성과: 
  - TableView를 사용해 페이지를 처음 구성해봄
  - ScrollView를 대체하여 TableView를 사용할 수 있다는것을 알게됨
  - API 호출을 할 수 있게 되었다.
